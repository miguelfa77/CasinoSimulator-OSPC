# Class structure

Customer class: 
Attributes:
- Name
- Age
- ankroll
- Loyalty level (e.g., bronze, silver, gold)

Methods:
- bet(amount)
- win(amount)
- lose(amount)
- is_bankrupt()
- get_loyalty_level()
- go_bathroom()

HighSpender class:
Inherits from Customer class
Attributes:
- Minimum bet amount
- Maximum bet amount
Methods:
- bet(amount) - Overrides the Customer class's Bet() method to be higher.
- enter_vip()

MediumSpender class:
Inherits from Customer class
Attributes:
- Minimum bet amount
- Maximum bet amount
Methods:
- bet(amount) - Overrides the Customer class's Bet() method.
- enter_vip()

LowSpender class:
Inherits from Customer class
Attributes:
- Minimum bet amount
- Maximum bet amount
Methods:
- bet(amount) - Overrides the Customer class's Bet() method to be lower.
- enter_vip()

Dealer class -> Miguel:
Attributes:
- Name
- Age
Methods:
- deal_cards()
- take_bets()
- pay_winners()
- collect_losses()

Table class: 
Attributes:
- Game type (blackjack, poker, roulette)
- Minimum bet amount
- Maximum bet amount
- List of players
Methods:
- add_player(player)
- remove_player(player)
- start_game()
- end_game()

BlackJackTable class: 
Inherits from Table class
Methods
- start_game()
- is_game_over() / end_game()

PokerTable class: (CASTRO & ANA)
Inherits from Table class
Methods
start_game()
get_winner() / end_game()

RouletteTable class:
Inherits from Table class
Methods
start_game()
get_outcome() / end_game()

Bar class:
Attributes:
- List of drinks
- Price of each drink
Methods:
- serve_drink(drink, customer)
- get_drink_price(drink)

- Age

Casino class:
Attributes:
- OpeningTime
- ClosingTime
- ListOfEmployees
- ListOfTables
Methods:
- open()
- close()
